BeamerPresenter
===============

This software provides a minimalist presenter mode for pdf presentations and more specifically beamer presentations.

Requirements:
- Windows;
- `SumatraPDF.exe` registered in `PATH` environment variable;
- SumatraPDF tabs disabled.
